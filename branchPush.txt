1. Создать ветку - git branch name, создать и сразу переключится – git checkout -b name.
2. Переключаться между коммитами/ветками – git checkout (название ветки или id коммита)
3. Изменить текст коммита – git commit --amend (откроется редактор, в котором можно изменить коммит), git commit --amend -m “Text new commit” (коммит изменится сразу с терминала)
4. Запушить ветку на github – git push
5. Вернуть состояние файла = последнему коммиту – git restore file.txt
